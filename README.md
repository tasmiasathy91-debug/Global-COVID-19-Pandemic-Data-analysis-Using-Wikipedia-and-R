# Analysis of Global COVID-19 Pandemic Data Using Wikipedia and R

# Overview

This project successfully demonstrates how to collect and analyze real-world COVID-19 pandemic data using web scraping techniques in R.

By leveraging HTTP requests, HTML parsing, and data preprocessing methods, valuable insights is obtained regarding global testing efforts, infection prevalence, and country-level comparisons.

The analysis highlightes the importance of testing data in understanding pandemic trends and demonstrates practical skills in web scraping, data cleaning, exploratory data analysis, and statistical interpretation.
